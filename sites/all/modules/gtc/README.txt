-- INTRODUCTION --

The Generate translated content module, Adds the ability to Devel module that
generate localized content.

It adds a checkbox which have `Generate translated contents` title to the
`generate content` form (admin/config/development/generate/content).

-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.

* Note: You should import your selected language translation, in core or at
least one of your modules before using this module on interface > Import
(admin/config/regional/translate/import).
